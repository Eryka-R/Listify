# Listify 

**Listify** es una aplicación Android que permite crear, editar e importar listas de ítems, seleccionar elementos y exportarlos como texto separado por comas para copiar o compartir fácilmente.

La app está pensada como una herramienta **genérica**, no solo para listas de la compra: puede usarse para tareas, viajes, ideas, inventarios, etc.

---

## Funcionalidades

### Gestión de listas
- Crear nuevas listas con nombre personalizado
- Ver todas las listas creadas
- Cambiar nombre de una lista 
- Duplicar listas
- Eliminar listas

### Edición de listas
- Añadir ítems manualmente
- Sugerencias automáticas basadas en ítems usados anteriormente
- Marcar / desmarcar ítems
- Eliminar ítems 

### Importar desde texto
- Crear una lista a partir de texto separado por comas o saltos de línea  
  Ejemplo: Yogurt, leche, pan
- Asignar nombre a la lista importada

### Exportar
- Exportar los ítems seleccionados como texto:
- Copiar al portapapeles
- Compartir mediante apps del sistema (WhatsApp, Telegram, email, etc.)

###  Persistencia
- Los datos se guardan automáticamente en almacenamiento interno
- Las listas se conservan aunque se cierre la aplicación
